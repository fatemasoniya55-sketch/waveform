📡 Digital Signal Encoding Waveform Generator

⚡📊
🔌 Signal voltage definition: +V , 0 , –V
All waveforms display normalized levels: “+V” = positive voltage (e.g., +5V), “0” = zero volts (ground), “–V” = negative voltage (e.g., –5V). Encoding schemes use these to represent binary 0/1 — unipolar uses +V/0, polar/bipolar uses all three.
🔴 +V   High ⚫ 0   Ground 🔵 –V   Negative
